# fkwechatzan
微信集赞、评论插件，随意设置点赞数量和评论。集赞换礼品终结杀手！

### Install

- 如果你会iOS越狱开发的话，应该装有theos环境，那么直接运行build.sh脚本就可以了。
- 如果你会安装deb插件包的话，package目录下有一个deb包。自己传到手机上手动安装也行。
- 以上你都不熟悉的话，那就等两天插件过bigboss审核以后，直接在cydia里面搜索fkwechatzan安装就行
